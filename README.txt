Práctica final para la asignatura de programación multimedia y dispositivos móviles.

Esta práctica se trata de un sencillo gestor de tareas. Este permite añadir, eliminar y modificar tareas.

¿QUÉ INCLUYE?
- Uso de glide para poner imagenes mediante url a las tareas.
- Recursos string, dimen y colors.
- Uso del binding.
- Intents para las transiciones.
- Melodía con el correcto ciclo de vida en mainactivity (la pantalla donde se muestran las tareas).
- Uso de shared preferences para el login .
- SQL Lite mediante Room y corrutinas, usando las funciones query, insert, update y delete.
- Fragments.

ANOTACIONES
- Puedes borrar una tarea manteniendola presionada.
- Si tienes problemas con los paddings, margins... prueba en tu móvil.
